# spit-network-cdn

Public image host for [@spitnetwork](https://instagram.com/spitnetwork) post graphics.

This repo only holds rendered post PNGs (no source, no secrets). Used as a free CDN for Buffer's bulk-upload — raw GitHub URLs are public and reliable.

Naming: `<pillar>/<content_id>.png` (e.g. `FO/FO055.png`).

Source database, captions, and templates live in the private `holdco-vault` repo.
